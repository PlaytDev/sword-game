# ⚔ 검 강화하기

50강까지 도전하는 검 강화 시뮬레이션 게임

## 🎮 플레이하기
**👉 [지금 바로 플레이](https://playtdev.github.io/sword-game/)**

## 📖 게임 소개
낡은 단검부터 전설의 엔딩의 검까지, **50강**의 강화 단계가 기다립니다.

- 100만원의 자본으로 시작
- 강화에 실패하면 검이 **파괴**됩니다
- **방지권**으로 파괴를 막을 수 있습니다 (등급이 높아질수록 더 많이 필요)
- **워프권**으로 특정 등급까지 즉시 도달 가능 (확정)
- 51개 검 이미지를 직접 커스터마이징 가능

## ✨ 주요 기능
| 기능 | 설명 |
|---|---|
| 강화 시스템 | 50강까지 도전, 등급별 성공률 차등 |
| 방지권 | 1개/10개/100개 묶음 판매 |
| 워프권 | 5강 ~ 45강 즉시 워프 (확정) |
| 자동 저장 | 자본/방지권/검 등급 브라우저에 자동 저장 |
| 이미지 커스터마이징 | 51개 검 슬롯에 원하는 이미지 등록 |
| 다크 모드 | OS 설정에 따라 자동 적용 |

## 🎯 게임 팁
- 0~9강은 비교적 안전하므로 자본을 빠르게 늘릴 수 있습니다
- 15강부터는 방지권 없이 도전하지 마세요
- 워프권은 시간 단축용 — 직접 강화의 약 2배 비용이지만 확정이라 안전합니다
- 파괴 시 자본이 500원 미만이면 자동으로 500원까지 채워줍니다
